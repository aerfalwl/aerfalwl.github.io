Leetcode 532. K-diff Pairs in an Array

发表于 2020-10-31 分类于 leetcode 阅读次数：

题目描述

英文题目

中文题目

难度：中等

题意

说明：题目来自leetcode中文官网

给定一个整数数组和一个整数 k，你需要在数组里找到不同的 k-diff 数对，并返回不同的 k-diff 数对 的数目。

这里将 k-diff 数对定义为一个整数对 (nums[i], nums[j])，并满足下述全部条件：

0 <= i, j < nums.length
i != j
|nums[i] - nums[j]| == k
注意，|val| 表示 val 的绝对值。
