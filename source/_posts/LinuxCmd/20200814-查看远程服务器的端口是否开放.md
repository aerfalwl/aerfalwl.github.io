# 查看远程服务器的端口是否开放

## 1. nc（netcat）

nc是一个简单的Unix工具，使用TCP或者UDP协议读写网络连接上的数据。

```bash
nc [OPTIONS...] [hostname] [port]
```

如果未开放，则会显示请求被拒绝。例如：

```bash
nc node1 9100
# Ncat: Connection refused.
```

如果指定端口开放，则会显示连接成功。例如：

```bash
nc node1 9100
# Ncat: Connection succeeded
```

## 2. telnet

命令格式

```bash
telnet [HostName or IP] [PortNumber]
```
