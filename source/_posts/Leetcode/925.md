## 题目描述

[英文题目](https://leetcode.com/problems/long-pressed-name/)

[中文题目](https://leetcode-cn.com/problems/long-pressed-name/)

难度：简单

**代码如下：**

```java
class Solution {
    public boolean isLongPressedName(String name, String typed) {
        if (null == name || null == typed) return false;
        int lenN = name.length();
        int lenT = typed.length();
        int n = 0, t = 0;
        
        while (n < lenN && t < lenT) {
            char cn = name.charAt(n);
            char ct = typed.charAt(t);
            if (cn == ct) {
                n++;
                t++;
                continue;
            }
            if (t == 0) return false;
            char pre = typed.charAt(t - 1);
            if (pre == ct) {
                while (pre == ct) {
                    t++;
                    if (t >= lenT) break;
                    ct = typed.charAt(t);
                }
            } else {
                return false;
            }
        }
        if (n != lenN) return false;
        while (t < lenT) {
            char ct = typed.charAt(t);
            if (ct != name.charAt(n - 1)) {
                return false;
            }
            t++;
        }
        return true;
    }
}
```
